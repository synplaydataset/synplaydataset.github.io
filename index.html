Dataset Download

There are 73,892 images with more than 6.5M human instances in the entire set. The dataset contains 6 different games and each game is zipped into a tar.xz file. The ground truth annotations for human instances (bounding boxes, pixel-level labels) and all the meta data (camera location, pose, etc.) for each game are included in each game file.
